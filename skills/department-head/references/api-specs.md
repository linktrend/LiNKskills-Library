# API & Tool Technical Specifications

## Inputs
- Stream-level `PROGRESS.md` files from Lead Engineer and QA instances.
- Studio objective and current priority window.

## Outputs
- COO directive list with owner, priority, and due logic.
- Consolidated risk flags and unblock plan.

## Governance Requirement
- Every directive must map to at least one observed status/blocker from `PROGRESS.md`.
