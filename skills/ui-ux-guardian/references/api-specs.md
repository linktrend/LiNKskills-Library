# API & Tool Technical Specifications

## Playwright Tooling
- `playwright-cli` for deterministic screenshot and export operations.
- `fast-playwright` for interactive accessibility and session-based navigation.

## Audit Inputs
- Target URL/page list.
- Baseline artifact path.
- Visual diff threshold and token policy.

## Audit Outputs
- Pass/fail summary.
- Per-page diff metrics.
- Remediation checklist.
